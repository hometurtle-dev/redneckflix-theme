# 🎬 RedneckFlix Theme

> A dark red, Netflix-inspired CSS theme for [Jellyfin](https://jellyfin.org) — with glowing red accents.

![Theme](https://img.shields.io/badge/Jellyfin-Theme-e50914?style=flat-square&logo=jellyfin&logoColor=white)
![Version](https://img.shields.io/badge/version-1.0.0-e50914?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-555?style=flat-square)

---

## ✨ Features

- 🖤 Background ultra-sombre (Netflix `#141414`)
- 🔴 Rouge `#e50914` comme couleur principale
- ✨ Effets glow rouges sur les éléments interactifs (cartes, boutons, nav, player)
- 🎞️ Cartes avec zoom + bordure rouge glowy au survol
- 🎮 Player OSD : barre de progression rouge glowy
- 🔠 Nom **RedneckFlix** mis en avant dans le header (+ emplacement logo prêt)
- 📱 Compatible desktop & mobile

---

## 🚀 Installation

1. Ouvre Jellyfin
2. **Tableau de bord → Général → CSS personnalisé**
3. Colle cette unique ligne :

```css
@import url("https://raw.githubusercontent.com/hometurtle-dev/redneckflix-theme/main/theme.css");
